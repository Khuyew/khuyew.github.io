🚀 Начало работы

KHAI Assistant — это мощный ИИ-помощник с множеством функций для работы с текстом, изображениями и голосом.

Быстрый старт

Чтобы начать использовать KHAI Assistant:

  1. Откройте приложение в браузере
  2. Начните диалог, введя сообщение в поле ввода
  3. Используйте кнопки для прикрепления файлов или изображений
  4. Выберите подходящую модель ИИ для ваших задач

🎨 Интерфейс приложения

KHAI Assistant имеет интуитивно понятный интерфейс, разделенный на несколько основных областей:

Область чата

Основное пространство для отображения диалога с ИИ

Поле ввода

Ввод сообщений, прикрепление файлов и управление режимами

Боковая панель

Управление чатами, настройки и информация о приложении

Мини-карта

Быстрая навигация по длинным диалогам

🤖 Модели искусственного интеллекта

KHAI Assistant поддерживает несколько современных моделей ИИ:

Доступные модели

  • GPT-5 Nano — Быстрая и эффективная модель для повседневных задач
  • O3 Mini — Продвинутая модель с улучшенными возможностями рассуждения
  • DeepSeek Chat — Универсальная модель для общения и решения задач
  • Gemini 2.0 Flash — Новейшая быстрая модель от Google
  • Grok Beta — Модель от xAI с уникальным характером

Выбор модели

Для смены модели:

  1. Нажмите на кнопку с иконкой мозга в заголовке
  2. Выберите нужную модель из списка
  3. Подтвердите выбор

💬 Чат с искусственным интеллектом

Основная функция KHAI Assistant — интеллектуальный диалог с ИИ.

Отправка сообщений

Вы можете отправлять сообщения несколькими способами:

  • Наберите текст и нажмите Enter
  • Используйте кнопку отправки
  • Воспользуйтесь голосовым вводом

Форматирование текста

ИИ понимает и генерирует текст с форматированием:

**Жирный текст**
*Курсив*
`код`
```python
print("Блок кода")
```

📁 Анализ файлов

KHAI Assistant может анализировать содержимое различных файлов.

Поддерживаемые форматы

Текстовые файлы

TXT, MD, HTML, CSS, JSON

Файлы кода

Python, Java, C++, C#, PHP, Ruby, Go

Данные

XML, CSV, YAML, YML

Как использовать

  1. Нажмите кнопку "+" в поле ввода
  2. Выберите файл для загрузки
  3. Добавьте текстовый запрос (опционально)
  4. Отправьте сообщение
